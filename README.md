The Russia–Ukraine conflict has become one of the most widely discussed geopolitical issues on social media platforms, including YouTube. User comments on the platform reflect diverse public opinions, making sentiment analysis essential for understanding public perceptions of the conflict. This study aims to implement the IndoBERT model to classify the sentiment of Indonesian-language YouTube comments related to the Russia–Ukraine conflict and evaluate the model’s performance in identifying positive, negative, and neutral sentiments. The dataset consists of 4,828 comments collected from six YouTube videos between November 14, 2025, and January 10, 2026. After preprocessing and removing empty data, 4,758 comments remained and were labeled using a lexicon-based approach with the InSet Lexicon with a positive sentiment distribution of (18.31%), negative (64.10%), and neutral (17.59%). The preprocessing stages included case folding, cleaning, normalization, tokenization, stopword removal, and stemming. The IndoBERT model was then fine-tuned and evaluated using accuracy, precision, recall, and F1-score metrics. The results indicate that the best performance was achieved using a 70:15:15 data split with a batch size of 16 and a learning rate of 2e-5, resulting in an accuracy of 74.79%, with precision, recall, and F1-score values exceeding 70%. Furthermore, the application of oversampling and undersampling techniques did not improve performance compared to the baseline method. Based on these findings, it can be concluded that IndoBERT is capable of performing sentiment analysis on Indonesian-language YouTube comments related to the Russia–Ukraine conflict with good performance and is effective in understanding the context of the Indonesian language.
